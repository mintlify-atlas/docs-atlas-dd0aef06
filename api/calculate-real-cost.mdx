## Overview

The `calculate_real_cost` tool converts a product price into meaningful real-world equivalents (ramen packs, coffees, hours of work, Netflix subscriptions) and calculates what the money could be worth if invested instead. This helps users understand the true impact of their spending decisions.

**Tool name:** `calculate_real_cost`

## When to use this tool

Use this tool when:
- Users want to understand what a price really means in practical terms
- Users ask "what could I buy instead" or "put this in perspective"
- Users mention "show me what I could do with this money"
- You need to provide context for a purchase decision

<Note>
After calling this tool, always render a `spending-insight` component with the returned data. Never display raw tool output or JSON as text.
</Note>

## Output schema

<ResponseField name="product" type="string">
  The name of the product being analyzed.
</ResponseField>

<ResponseField name="price" type="number">
  The price in USD.
</ResponseField>

<ResponseField name="equivalents" type="array">
  Array of 4 real-world price equivalents showing what else you could buy.
  
  <Expandable title="Equivalent object structure">
    <ResponseField name="item" type="string">
      The name of the equivalent item (e.g., "Large Coffees", "Instant Ramen Packs").
    </ResponseField>
    
    <ResponseField name="count" type="number">
      How many of this item you could buy for the same price.
    </ResponseField>
    
    <ResponseField name="emoji" type="string">
      A relevant emoji for the item (e.g., "☕", "🍜").
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="monthlyIfRecurring" type="string">
  Formatted string showing annual cost if this purchase was made monthly (e.g., "$4,176/year").
</ResponseField>

<ResponseField name="invested10yr" type="string">
  Formatted string showing what this amount would be worth if invested at 7% annual return for 10 years (e.g., "~$685 if invested").
</ResponseField>

<ResponseField name="verdict" type="string">
  A context-aware message based on the price tier:
  - Over $500: "This is a significant purchase. Sleep on it."
  - $100-$500: "Not trivial. Ask yourself: will you care about this in a month?"
  - Under $100: "Small but they add up. Death by a thousand cuts."
</ResponseField>

## Implementation details

The tool uses preset conversion rates:

| Item | Cost per unit | Emoji |
|------|--------------|-------|
| Instant Ramen Packs | $0.50 | 🍜 |
| Large Coffees | $5.50 | ☕ |
| Hours of Min Wage Work | $7.25 | 👷 |
| Netflix Months | $15.49 | 📺 |

Investment calculation uses a 7% annual return rate compounded over 10 years:
```typescript
invested10yr = price × (1.07)^10
```
