      Get WalletWake running locally in minutes
    </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the WalletWake repository to your local machine:

      ```bash
      git clone https://github.com/sivaratrisrinivas/walletwake.git
      cd walletwake
      ```
    </Step>

    <Step title="Install dependencies">
      Install the required packages using npm:

      ```bash
      npm install
      ```
    </Step>

    <Step title="Configure environment variables">
      Create a `.env.local` file in the project root with your API keys:

      ```env
      NEXT_PUBLIC_TAMBO_API_KEY=your_tambo_key
      GEMINI_API_KEY=your_google_ai_studio_key
      ELEVENLABS_API_KEY=your_elevenlabs_key
      ELEVENLABS_VOICE_ID=pNInz6obpgDQGcFmaJgB
      ```

      <Tip>See the [API Keys](/configuration/api-keys) page for details on obtaining these credentials.</Tip>
    </Step>

    <Step title="Start the development server">
      Launch the app in development mode:

      ```bash
      npm run dev
      ```

      Open [http://localhost:3000](http://localhost:3000) in your browser. Click **Start Chatting** and ask about any product.
    </Step>
  </Steps>
