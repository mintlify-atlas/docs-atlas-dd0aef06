WalletWake is a fully chat-driven impulse buy intervention app powered by Tambo's Generative UI. Tell the AI what you want to buy, and it shows you a funeral for that purchase—complete with guilt math, a 48-hour waiting period, and a spoken eulogy.

<Note>
This quickstart assumes you have Node.js 20+ and npm installed. If you need help setting up your environment, see the [Installation](/installation) guide.
</Note>

## Get started in 5 steps

<Steps>
  <Step title="Clone the repository">
    Clone the WalletWake repository to your local machine:

    ```bash
    git clone https://github.com/yourusername/walletwake.git
    cd walletwake
    ```
  </Step>

  <Step title="Install dependencies">
    Install all required packages using npm:

    ```bash
    npm install
    ```

    This installs the core dependencies:
    - **Next.js 16** — React framework
    - **@tambo-ai/react** — Generative UI components
    - **@google/genai** — Google Gemini for text and image generation
    - **GSAP** — Smooth animations for the funeral carousel
    - **Zod** — Schema validation for Tambo tools
  </Step>

  <Step title="Configure environment variables">
    Create a `.env.local` file in the project root with your API keys:

    ```env
    NEXT_PUBLIC_TAMBO_API_KEY=your_tambo_key
    GEMINI_API_KEY=your_google_ai_studio_key
    ELEVENLABS_API_KEY=your_elevenlabs_key
    ELEVENLABS_VOICE_ID=pNInz6obpgDQGcFmaJgB
    ```

    <Tip>
    Get your API keys from:
    - **Tambo**: [tambo.ai](https://tambo.ai)
    - **Google Gemini**: [ai.google.dev](https://ai.google.dev)
    - **ElevenLabs**: [elevenlabs.io](https://elevenlabs.io)
    </Tip>

    <Warning>
    Never commit `.env.local` to version control. It's already in `.gitignore` by default.
    </Warning>
  </Step>

  <Step title="Start the development server">
    Run the development server:

    ```bash
    npm run dev
    ```

    Your app will be available at [http://localhost:3000](http://localhost:3000).
  </Step>

  <Step title="Make your first intervention">
    Open your browser and try these interactions:

    1. Click **Start Chatting** to open the chat overlay
    2. Type: "I want a PS5"
    3. The AI responds with a **Funeral Ticket** showing the real cost
    4. Click **Enter Funeral Mode** to see:
       - Price converted to ramen packs, coffees, and work hours
       - A custom gloomy image generated by Gemini
       - A 48-hour countdown timer
       - GSAP-animated carousel of cheaper alternatives
       - A spoken eulogy (turn on sound)

    <Tip>
    In development, you can click **Skip 48h** in the bottom right to fast-forward to the "Clarity" screen.
    </Tip>
  </Step>
</Steps>

## What you just built

Your WalletWake instance includes:

### 5 Tambo generative components

- **funeral-ticket** — Warning ticket for general purchase intent
- **roast-receipt** — Sarcastic receipt with itemized breakdown
- **spending-insight** — Real-world price equivalents (ramen, coffee, etc.)
- **impulse-score** — Visual danger gauge rating purchases 1-10
- **alternative-suggestion** — List of cheaper/better options

### 2 Tambo local tools

- **calculate_real_cost** — Converts prices to ramen/coffee/work equivalents
- **search_products** — Searches the mock product catalog

### Context-aware AI agent

The agent has a witty, sarcastic personality that:
- Discourages impulse purchases using humor
- Provides brutally honest financial advice
- Responds in natural, conversational language
- Never shows raw JSON or tool output
